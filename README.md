# 과제 #2 

## 주제: top ps, jobs, kill 명령어 조사 

#

## 1.명령어 한눈 정리
| 명령어 | 기능 | 특징 |
|--------|------|------|
| top    | 실시간 프로세스 모니터링 | CPU/메모리 실시간 변화 확인 |
| ps     | 프로세스 목록 출력 | CPU/메모리 실시간 변화 확인
| jobs | 작업 목록 확인 | 백그라운드 작업 관리 |
| kill | 프로세스 종료 | PID로 프로세스 제어 |

#

## 2-1 top 명령어란

### 1. top 명령어란?
- **실시간 시스템 모니터링 도구**예요.
- CPU, 메모리, 프로세스 상태를 확인할 수 있어요.

### 2. 화면 구성 
`top` 화면은 크게 3부분으로 나눌 수 있어요.
1. 시스템 정보 (상단)
- **uptime** : 시스템 켠 지 얼마나 됐는지
- **tasks** : 전체 프로세스 수
- **load average** : CPU 사용률 평균
2. 프로세스 리스트 (하단)
- **PID** : 프로세스 ID
- **USER** : 프로세스 실행 사용자
- **%CPU** : CPU 사용률
- **%MEM** : 메모리 사용률
- **COMMAND** : 실행한 프로그램 이름
- **%CPU, %MEM** : CPU와 메모리 사용률

### 3. 자주 쓰는 단추키
| 조작 키 | 기능 | 설명 |
| ---- | ------------ | ------------------------- |
| `P` | CPU 사용량 순 정렬 | CPU를 많이 쓰는 프로세스가 위쪽에 표시됨 |
| `M` | 메모리 사용량 순 정렬 | 메모리 많이 쓰는 프로세스가 위쪽에 표시됨 |
| `k` | 특정 PID 종료 | 종료할 프로세스의 PID를 입력하여 종료 가능 |
| `q` | 종료 | `top` 종료하고 쉘로 돌아감 |

### 4. 실전 예제
1. CPU 많이 쓰는 프로세스 확인:
- ```top → P```
2. 메모리 많이 쓰는 프로세스 확인:
- ```top → M```
3. 특정 PID 종료:
- ```top → k → PID 입력```

## 2-2 ps 명령어

### 1. ps 명령어란?
- **현재 실행 중인 프로세스 목록을 보는 명령어**예요.
- `top`이 실시간 모니터링이라면, `ps`는 순간 상태를 보여주는 스냅샷이라고 생각하면 돼요.
- 기본적으로 자기 터미널에서 실행한 프로세스만 보여줘요.

### 2. 자주 쓰는 옵션
| 명령어 | 설명 |
| ------------------ | -------------------------------------- |
| `ps -e` 또는 `ps -A` | 시스템 전체 프로세스 보기 |
| `ps -f` | 전체 정보(full) 보기 (UID, PID, PPID, CMD 등) |
| `ps -u 사용자명` | 특정 사용자의 프로세스 보기 |
| `ps aux` | 많이 쓰이는 전체 프로세스 보기 방법 |
| `ps -ef` | 다른 형태의 전체 프로세스 보기 |

+ !!! `ps aux`와 `ps -ef`는 거의 비슷하지만 출력 형식이 조금 달라요.

### 3. 출력 컬럼 의미 (대표적)

`SER` : 프로세스를 실행한 사용자

`PID` : 프로세스 ID

`%CPU` : CPU 사용률

`%MEM` : 메모리 사용률

`VSZ` : 가상 메모리 사용량 (KB 단위)

`RSS` : 실제 메모리 사용량 (KB 단위)

`TTY` : 터미널 종류

`STAT` : 프로세스 상태 (R: 실행, S: 대기, Z: 좀비 등)

`COMMAND` : 실행한 명령어

### 4. 실전 예제
1. 내가 실행한 프로세스 보기:
- `ps`
2. 모든 사용자 프로세스 보기:
- ```ps -ef```
3. 특정 사용자 프로세스 보기:
- ```ps -u 사용자명```
1. CPU 사용률 높은 순으로 정렬:
- ```ps aux --sort=-%cpu```
2. 메모리 사용률 높은 순으로 정렬:
- ```ps aux --sort=-%mem```
