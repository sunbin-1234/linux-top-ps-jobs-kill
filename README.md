# 과제 #2 
## 주제: top ps, jobs, kill 명령어 조사 
#
## 1.명령어 한눈 정리
| 명령어 | 기능 | 특징 |
|--------|------|------|
| top    | 실시간 프로세스 모니터링 | CPU/메모리 실시간 변화 확인 |
| ps     | 프로세스 목록 출력 | CPU/메모리 실시간 변화 확인
| jobs | 작업 목록 확인 | 백그라운드 작업 관리 |
| kill | 프로세스 종료 | PID로 프로세스 제어 |
#
## 2-1 top 명령어
### 실시간 프로세스 모니터링
1. **용도**:
- 현재 실행 중인 프로세스와 시스템 상태(CPU, 메모리 등)를 실시간으로 확인.
2. **주요 특징**:
- CPU 사용량, 메모리 사용량, 실행 중인 프로세스 목록을 실시간 업데이트
- 프로세스를 직접 종료할 수도 있음
### top 사용 예시
| 조작 키 | 기능 | 설명 |
| ---- | ------------ | ------------------------- |
| `P` | CPU 사용량 순 정렬 | CPU를 많이 쓰는 프로세스가 위쪽에 표시됨 |
| `M` | 메모리 사용량 순 정렬 | 메모리 많이 쓰는 프로세스가 위쪽에 표시됨 |
| `k` | 특정 PID 종료 | 종료할 프로세스의 PID를 입력하여 종료 가능 |
| `q` | 종료 | `top` 종료하고 쉘로 돌아감 |
## 2-2 ps 명령어
### 현재 프로세스 확인
1. **용도**:
- 현재 시스템에서 실행 중인 프로세스를 스냅샷 형태로 확인.
2. **주요 옵션**:
- `ps aux` : 모든 프로세스를 자세히 보기
- `ps -ef` : 표준 UNIX 스타일로 프로세스 확인
